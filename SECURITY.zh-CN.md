# 安全政策

[English](SECURITY.md) | [简体中文](SECURITY.zh-CN.md)

## 支持版本

`main` 分支是当前受支持的开发版本。

## 报告安全问题

请通过 GitHub Private Security Advisory 或维护者提供的私密渠道报告安全问题。

不要在公开 Issue 中发布密钥、Token、私有语料样本、个人信息或专有模型路径。

## 数据与模型安全

LocalTune Console 面向本地微调，使用者需要自行负责：

- 核实下载模型权重的许可证。
- 核实训练数据的许可证、来源和授权范围。
- 避免将私有数据加入版本库或公开分享。
- 让无登录认证的管理台只监听本机，或自行配置访问控制。
- 仅从可信来源加载启用了 `trust_remote_code` 的模型。
- 在重新发布模型或输出前进行必要审查。
